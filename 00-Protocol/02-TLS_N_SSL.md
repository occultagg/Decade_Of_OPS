# TLS 和 SSL

都用于HTTPS.

SSL是早期版本,TLS是基于SSL3.0,在安全性,性能和兼容上的改进版本.

现代web应用几乎都使用TLS.

目前推荐配置的都是TLS1.2或更高版本.

# TLS1.3

2018年发布,最新主要版本.

- 将握手的两个往返(RTT)减少到1个,甚至支持0-RTT(零往返时间回复),显著降低延迟,提高页面加载速度.
- 加密算法优化: 移除所有已知不安全的加密算法
- 握手消息加密:server hello后,所有握手消息(包括证书)都被加密
- 协议简化: 去除多余功能
- 性能提升: 整体加密更快
